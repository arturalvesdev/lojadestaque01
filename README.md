# Destaque premium — Loja online

Site da loja Destaque premium (Next.js).

## Começar rápido

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000)

## Documentação

- [Configuração (Supabase e login)](docs/CONFIGURACAO.md)
- [Estrutura do projeto](docs/ESTRUTURA-DO-PROJETO.md)

## Conta e sacola

- `/conta` — entrar ou cadastrar (e-mail e senha)
- Sacola exige login
- Produtos exigem tamanho e cor antes de adicionar
